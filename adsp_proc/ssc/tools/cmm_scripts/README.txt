==============================================================================
  Introduction
==============================================================================

These scripts are a set of CMM files designed to parse SLPI ramdumps that have
been loaded into T32. The parsing scripts will attempt to walk through the
ramdump and print out the SEE framework state along with any other potentially
useful data structures.


==============================================================================
  Usage
==============================================================================

In the T32 window, execute the following command:
  CD.DO <SLPI_BUILD_ROOT_DIR>\slpi_proc\ssc\tools\cmm_scripts\ssc_parser.cmm
